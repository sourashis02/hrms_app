# hrms_app
This is a hrms app

# Steps to run the these app locally


* Run the Node.js backend app
1. cd hrms_backend
2. npm install
3. npm start
4. You can access the app at http://localhost:3000

* Run the React frontend app
1. cd hrms_frontend
2. npm install
3. npm run dev
4. You can access the app at http://localhost:5173
